## Building from Source

1. Clone the repository:

   ```bash
   git clone https://github.com/XDPXI/XDs-Code.git
   ```

2. Navigate to the project directory:

   ```bash
   cd XDs-Code
   ```

3. Install dependencies:

   ```bash
   pnpm install
   ```

4. Build the project:

   ```bash
   pnpm run build
   ```
